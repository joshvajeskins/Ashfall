# Prompt 6: Sound Effects - Download List

## Goal
Download sound effects for game feedback. All sounds should be short (0.5-3 seconds) MP3 files.

## After Adding Audio

The game will automatically load and play sounds. If a file is missing, the SoundManager silently skips it (no errors).

Test by:
1. Running the game
2. Entering combat - should hear attack/hit sounds
3. Picking up items - should hear pickup chime
4. Dying - should hear death sound
