# Prompt 5: Pixel Art Assets - AI Generation Prompts

## Goal
Generate all pixel art assets for the game using AI image generators (DALL-E, Midjourney, Stable Diffusion, etc.)

## Asset Style Guide
- **Style:** 16-bit pixel art, dark fantasy theme
- **Color palette:** Dark browns, grays, deep reds, muted purples
- **Background:** Transparent (PNG) for sprites, solid for tiles
- **Resolution:** Characters/tiles 32x32, items 24x24, boss 64x64

## Generation Tips

1. **For Midjourney:** Add `--v 6 --style raw` for pixel art consistency
2. **For DALL-E:** Be explicit about "32x32 pixels" and "transparent background"
3. **For Stable Diffusion:** Use pixel art LoRAs for best results
4. **Post-processing:** May need to resize/crop to exact dimensions in Photoshop/GIMP
